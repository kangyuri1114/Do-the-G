# Do-the-G

### 제목 
Do the G !

### 스토리 라인
땅을 파다가 우연히 고구마의 맛을 알게 된 "도다지"씨.
" 고구마가 이렇게 맛있는 거구나..! "
도다지씨는 고구마를 찾아 떠난다.
그러나 고구마를 찾아 먹는 여정은 쉽지 않은데...

### 조작법
도다지씨는 계속 뛰어갑니다. 
마우스 왼쪽 클릭 시 고구마를 먹을 수 있습니다.
